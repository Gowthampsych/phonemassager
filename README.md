# PhoneMassager

## About PhoneMassager

**PhoneMassager** is an innovative app designed to utilize the vibration functionality of smartphones to offer a customizable massage experience. With its simple interface and adjustable settings, users can enjoy a personalized and convenient massage experience anywhere, directly from their phone. Whether for relaxation, pain relief, or just fun, PhoneMassager transforms your phone into a vibrating massager with just a tap!

### Features
- Multiple vibration intensity levels.
- Customizable vibration patterns.
- Easy-to-use, intuitive interface.
- Works on a wide range of Android devices.

---

## Installation Guide

### Prerequisites
- **Operating System**: Android 5.0 (Lollipop) or higher.
- **Storage Space**: Minimum 50MB of free space.
- **Permissions**: Vibration, storage access.

### Steps to Install
1. **Download the App:**
   - Visit the [GitHub repository for PhoneMassager](https://github.com/Gowthampsych/phonemassager) and download the latest release `.apk` file (if available).
   - Alternatively, build the app from source as described below.

2. **Build from Source (For Developers):**
   - **Clone the Repository**:
     ```bash
     git clone https://github.com/Gowthampsych/phonemassager.git
     cd phonemassager
     ```
   - **Open the Project**:
     Open the project in your preferred Android IDE (such as Android Studio).
   - **Install Dependencies**:
     Sync and download necessary dependencies using Gradle.
   - **Build and Run**:
     Build the project and run it on an Android emulator or a connected Android device using Android Studio.

3. **Install the App (Manual Installation)**:
   - Ensure "Install from unknown sources" is enabled on your device.
   - Locate and open the `.apk` file on your Android device.
   - Follow the prompts to complete the installation.

### Usage
1. Open the **PhoneMassager** app.
2. Select the desired vibration intensity and pattern.
3. Tap the "Start Massage" button to begin.
4. Adjust settings as needed for a personalized experience.
